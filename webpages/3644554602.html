Entry Level, students, novices, please not the following line:

"This is strictly for asthetics. No PHP coding necessary."

This is very misleading. You will always still have to code something unless someone is telling you that you will only need to comp out a design in photoshop or fireworks as a mockup or potentially slicing to export images for someone else to include in the site, YOU WILL BE CODING SOMETHING.

This is CSS...you still have to CODE CSS and you still have to know how to make the CSS talk to the HTML. Yes, it's a PHP web site, but PHP works like HTML, therefore you still have to have a working knowledge of how to get the CSS and PHP working together.

So, unless the rate being offered is hourly (and it's likely not), please be very careful of this.
